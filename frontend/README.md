# Festival of Hearts - Frontend

## Project Overview
Festivalul Inimilor (The Festival of Hearts) is a prestigious event balancing deep respect for tradition with a modern, exciting user experience.

## Visual Identity
- **Primary Color:** Carmine Red (#960018)
- **Secondary Color:** Canvas White (#F0EAD6)
- **Accent Color:** Harvest Gold (#E1AD01)
- **Dark Mode/Text:** Charcoal Navy (#181724)

## Site Structure
- Acasă (Homepage)
- Despre Festival (Concept & History)
- Program (Schedule & Locations)
- Ansambluri (Lineup/Participating Groups)
- Ediții Anterioare (Archive/History)
- Multimedia (Photo/Video Gallery)
- Înscriere (Applications for Volunteers/Ensembles)
- Info Utile & Contact (Location, FAQ, Press)

## Technical Features
- Language toggle (RO / EN / FR)
- Micro-animations
- Audio Player (Traditional music)
- Mobile First design
